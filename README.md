# Gameplay Portfolio

Live site: **https://simon936-git.github.io/Portfolio/**

React + TypeScript portfolio for junior gameplay programming roles.

## Deploy

Push to `main` with **GitHub Pages → Source: GitHub Actions**.

## Local dev

```bash
npm install
npm run dev
```
